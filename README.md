## MEANING OF NAME GONNECTION
Replacing C of Connection by G.
1. GO STADS FOR GOBOTICS.
2. G also stads for grief (a cause of misery), graveyard (referring to a scene of destruction), gale (a violent storm, type of disaster), grind (a harsh, difficult, or destructive experience),
   and gobsmacked (as a state of shock/disaster, though less direct).
